## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:Noroff-FEU-Assignments/js-frameworks-course-assignment-Relouding
```

2. Install the dependencies:

```
npm i next-axios
npm i next-link
npm i next-router
```

### Running

To run the app, run the following command:

```bash
npm run dev
```
